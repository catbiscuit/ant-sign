# AntSign

## 一、Fork 仓库

## 二、添加 Secret

**`Settings`-->`Secrets and variables`-->`Actions`-->`Secrets`-->`New repository secret`，添加以下secrets：**

| Name           | Secret                 |
| -------------- | ---------------------- |
| BARK_DEVICEKEY | IOS应用Bark 推送密钥   |
| BARK_ICON      | IOS应用Bark 推送的图标 |



## 三、运行

**`Actions`->`Run`->`Run workflow`**

## 四、查看运行结果

**`Actions`->`Run`->`build`**
